Upload these files/folders directly to the root of your GitHub repository:

index.html
3al_Saj_Menu.pdf
assets/
  cover.webp
  menu/page-01.webp ... page-12.webp

Important:
- Do not upload the ZIP itself to GitHub Pages.
- Extract the ZIP first, then upload the contents.
- GitHub Pages should publish from main / root.

Why this version works better on mobile:
- The old iframe PDF preview does not reliably display on mobile browsers.
- This version converts each PDF page into a responsive image.
- Customers can scroll the menu directly, and tap any page to open it full size.
- The original PDF is still included through the Open PDF button.
